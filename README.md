# Установка

Requirements:
1. rustc 1.39.0-nightly (97e58c0d3 2019-09-20)
2. exonum
3. fantom

For running nodes on your local machine
1. cd backend
2. cargo insstall --path .
3. ./launch.sh %number of nodes%
4. Open a browser at address stated in terminal
